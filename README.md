# noticed-bot
This bot can track all outgoing transactions on wallets, which you can specify in the "wallet.txt", you can also change the networks in the "config.network.js", so that the bot tracks all outgoing transactions only in the networks you need.  
You can also change "BOT_TOKEN" and "CHAT_ID" in the "bot.js", so that your own telegram bot sends you notifications!  
Type "npm i" at the command line, to download the required modules  
Type "npm run dev", to run the code
